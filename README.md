# 🎟️ EventFlow - Modern Event Registration Portal

A sleek, fully responsive event registration landing page built with pure Frontend technologies. This project is part of my professional portfolio, demonstrating my ability to handle user interactions, form validation, and dynamic UI updates without external frameworks.

## 🚀 Live Demo
[https://mightyleo247.github.io/event-registration-portal/]

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for accessibility and SEO.
* **CSS3:** Custom styling featuring Flexbox, Grid, and Glassmorphism effects.
* **Vanilla JavaScript:** DOM manipulation, Event Listeners, and LocalStorage integration.

## ✨ Key Features
* **Fully Responsive Design:** Optimized for mobile, tablet, and desktop views using Media Queries and CSS `clamp()`.
* **Dynamic Ticket Generation:** Once a user registers, the form is hidden and a unique digital ticket is generated dynamically using JS.
* **Data Persistence:** Uses `localStorage` to keep track of registration data locally.
* **Modern UI/UX:** Features a glassmorphism modal design and smooth CSS animations.
